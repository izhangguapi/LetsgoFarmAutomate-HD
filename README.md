# 星宝农场自动化（高清版）

该程序仅支持 Windows

游戏可无边框或窗口化运行

无边框需要屏幕比例为 16:9，

窗口化运行请前往**更多->设置->画面->分辨率**进行调整为下列分辨率：

3840×2160（4K 超高清）

2560×1440（2K 四倍高清）

1920×1080（1080p 全高清）‌

1280×720（720p 高清）

1600×900

1366×768

**运行该程序后请勿移动游戏窗口，会导致鼠标点击功能失效**
